Primitivas de WebGL com interação

Instruções:

  • Clique com o mouse em qualquer área do quadro para desenhar um ponto
  • Use a tecla "E" para alterar a espessura deste ponto
  • Use a tecla "K" para alterar a cor do ponto de maneira aleatória
  • Use a tecla "R" para traçar uma linha ao clicar em dois pontos
  • Use a tecla "T" para desenhar um triângulo ao clicar em três pontos
  • Use a tecla "C" para desenhar circunferências (clique para o centro)
